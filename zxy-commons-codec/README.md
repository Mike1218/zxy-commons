# zxy-commons-codec
加解密工具模块，包括RSA/BouncyCastle、base64、sha、md5等。
### 如果其他模块需要使用这些功能时，只需要在maven中加入：
```xml
依赖添加：
<dependency>
    <groupId>com.zxy</groupId>
    <artifactId>zxy-commons-codec</artifactId>
    <version>${zxy_commons_version}</version>
</dependency>
```

### 使用说明
#### MD5
MD5Utils

#### Base64
Base64Utils

#### SHA
SHAUtils

#### RSA
RSAUtils
